This library helps to create cnc milling programs for control panels.

On ebay are these parts easy to find, this gem helps to carve a control panel with such parts

- round rocker switches with a diameter of 20m are cheap (blue led)
- volt/amp meters
- speaker sockets (red/black) for 4mm banana plugs.

This gem can create a CNC program, for the desired tool diameter, see the example in the parent director to get started
